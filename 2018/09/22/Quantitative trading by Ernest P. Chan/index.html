Excess Returns = Portfolio Returns − Benchmark Returns
Information Ratio = Average of Excess Return / Standard Deviation of Excess Returns

3、交易费用影响收益

4、历史数据幸存者偏差

years?

避免常见的回测问题

1、Look-Ahead Bias
使用延迟的历史数据去计算交易机会

2、Data-Snooping Bias
过度拟合数据

3、Sample Size
交易时间（每年交易日的数量，每天交易小时数量、每天交易分钟数量）

4、Out-of-Sample Testing
将历史数据分为相等的两部分，一部分作为参数优化的数据，另外一份作为回测数据。（如果数据比较少的话，至少三分之一作为回测数据）

资金和风控管理

优化资金分配和杠杆

凯利公式：在一个期望收益为正的重复性赌局或者重复性投资中，每一期应该下注的最优比例。
假设胜的概率为p，败的概率为q=1-p，赔率为b，则每局最优下注比例为f*，这就是凯利公式：

    f* = (bp - q) / b

1、随着局数n的增大，按照凯利公式f*下注的资金Xn(f*)将远远超过按照任何其他比例f下注的资金Xn(f);
2、对于任何给定的目标资金额C，以凯利公式f*下注的策略超过该资金额所需要的期望时间（即期望局数）最少。

凯利公式在量化投资中的应用

将凯利公式应用于量化投资中确定投资品的最佳杠杆比例（仓位）

第一种：将所有盈利交易的平均收益除以所有亏损交易的平均亏损作为赔率。由于每个交易的开仓、平仓时间并不固定，每次交易的持续时间都是不同的。由于时间不同，交易的风险特性完全不同。所以这种不考虑时间的赔率计算方式是有问题。

第二种：利用凯利公式的思想，最大化单期对数收益率

假设一个投资品的单期的百分比收益率（即期末价格 / 期初价格 - 1）满足均值为 μ 、标准差为 σ 的正态分布。可以证明，在这个假设下，该投资品的单期对数收益率的期望为 μ - 0.5σ^2 。我们来看看最大化该对数收益率的杠杆率 L 是多少。
当我们使用  L 倍的杠杆时，均值和标准差分别变为 μL 和 σL ，因此对数收益率变为 μL - 0.5(σL)^2 。以 L 为自变量来最大化 μL - 0.5(σL)^2 。对其求一阶导数并使它为 0，并检查其二阶导数有：

一阶导数: μ-σ^2L=0
二阶导数: -σ^2

由一阶导数等于 0 可得最优的杠杆率为  L* = μ / σ^2 ，由于二阶导数恒小于 0，因此对数收益率在  L = L* 有最大值。因此，凯利公式确定的最优杠杆率就是：L* = μ / σ^2
在实际使用中， μ 和 σ 难以估计，此外不同期之间的收益率也很难保证绝对独立，因此业界普遍的观点是凯利公式的理论杠杆率风险较高。为此，普遍的做法是把  L* 看作是杠杆率的上限，而使用  L*/2 的杠杆率，这称之为“half-Kelly”。投资者可根据自己愿意承担的最大风险来决定是否进一步降低杠杆率。

凯利公式的核心是控制风险
